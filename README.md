
Інструкція користувача CryptoSpider
1. Запуск та Інтерфейс

Запустіть програму. Взаємодія відбувається через консольний інтерфейс, що забезпечує швидку роботу.
При першій спробі запуску Windows Defender може заблокувати запуск, бо пайтон скрипт вважається небезпечним, саме тому я додав код програми, щоб ви могли переконатися. При запуску .ЕХЕ файлу просто проігноруйте повідомленя і дозвольте запуск.

2. Вибір Алгоритму

У головному меню оберіть необхідний криптографічний алгоритм для дослідження: AES, Blowfish, Twofish або Kalyna.

3. Робота з Ключами

Введіть власний ключ шифрування або скористайтеся функцією генерації випадкових ключів для створення безпечного ключа для поточної демонстрації.

4. Процес Шифрування (Демонстрація)

Запустіть процес шифрування. Програма покаже виконання алгоритму крок за кроком (наприклад, етапи SubBytes чи ShiftRows для AES).

Використовуйте інтерактивні паузи , щоб зупинити процес на певному етапі та детально проаналізувати проміжні результати.

5. Аналіз Даних

Слідкуйте за кольоровими логами , які підсвічують зміни у даних для легшого сприйняття.

Переглядайте дані у зручному для вас форматі: як ASCII-текст або у бінарному представленні.

6. Дослідження (Лавинний ефект)

Змініть вхідний текст або ключ на один символ/біт, щоб спостерігати «Лавинний ефект» — програма візуалізує, як мінімальна зміна вхідних даних призводить до повної зміни шифротексту.
